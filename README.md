# Event Planner Website

A premium and modern Event Planner Website built with Next.js, TypeScript, Tailwind CSS, and shadcn/ui. The platform allows users to discover, explore, and book event services for birthdays, weddings, anniversaries, corporate events, and other special occasions.

## Features

### User Features
- Browse event categories
- View event packages and pricing
- Search and filter events
- Event gallery and portfolio
- User authentication (Login/Register)
- Online booking system
- Wishlist/Favorites
- Customer reviews and ratings
- Contact and inquiry forms
- Responsive design for all devices

### Admin Features
- Manage bookings
- Manage event packages
- Manage customers
- Manage reviews
- Dashboard analytics
- Content management
- User management

### Event Categories
- Birthday Celebrations
- Wedding Celebrations
- Anniversary Celebrations
- Engagement Ceremonies
- Baby Showers
- Corporate Events
- Housewarming Events
- Festival Celebrations
- Custom Events

## Getting Started

### Install
```
npm install
```

### Run the dev server
```
npm run dev
```

Then open http://localhost:3000

## Scripts
- `npm run dev`: start the dev server
- `npm run build`: create a production build
- `npm run start`: run the production server
- `npm run lint`: run lint checks

## Project Structure
- `app/`: route pages (App Router)
- `components/`: page sections and UI components
- `hooks/`: shared hooks
- `lib/`: utilities
- `public/`: static assets
